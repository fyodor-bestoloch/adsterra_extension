Расширение для поиска кодов конкуретнов на сайтах паблишеров



Гайд по установке:

1. Скачивайте папку с расширением (adsterra_extension) себе на устройство.
2. Заходите на chrome://extensions и выбирайте папку с расширением через “Загрузить распакованное расширение”. Готово, расширение должно появиться внутри браузера. Для использования необходимо открыть нужный сайт и нажать на иконку расширения в правом верхнем углу экрана. При возникновении проблем с отображением контента расширения рекомендуется открыть сайт заново в новой вкладке/новом окне.



---!!!---

При использовании необходимо оставлять всплывающее окно открытым. Удобнее всего открыть нужный сайт в новом окне, запустить расширение, альттабнуться и продолжать заниматься своими делами пока оно работает.
